# Using with Chart.js v2

If you want to use react-chartjs-2 with [Chart.js v2](https://www.chartjs.org/docs/2.9.4/), you should install [react-chartjs-2@v2](https://www.npmjs.com/package/react-chartjs-2/v/2.11.2):

```bash
yarn add react-chartjs-2@^2.11.2
```

or

```bash
npm install --save react-chartjs-2@^2.11.2
```
