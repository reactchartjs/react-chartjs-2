## Quickstart

قم بتثبيت هذه المكتبة مع تبعيات الأقران:

<علامات التبويب>
<قيمة العنصر = "الغزل" الافتراضي>

`` باش
إضافة الغزل chart.js react-chartjs-2
  ```

</TabItem>
<قيمة العنصر = "pnpm">

`` باش
pnpm إضافة chart.js react-chartjs-2
  ```

</TabItem>
<قيمة المنتج = "npm">

`` باش
تثبيت npm -- حفظ chart.js react-chartjs-2
  ```

</TabItem>
</علامات التبويب>

ثم قم باستيراد واستخدام المكونات الفردية:

`` jsx
استيراد {الرسم البياني كـ ChartJS و ArcElement و Tooltip و Legend} من "chart.js" ؛
استيراد {Doughnut} من "react-chartjs-2" ؛

ChartJS.register (ArcElement ، Tooltip ، Legend) ؛

<بيانات دونات = {...} />
```

لمعرفة المزيد عن استيراد وتسجيل العناصر انظر [شجرة تهتز] (https://react-chartjs-2.js.org/docs/migration-to-v4/#tree-hazing).
